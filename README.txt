Pour faire fonctionner correctement :
1) Cloner le repo
2) executer "make.bat" dans "bin"
3) ouvrir la solution dans "ide"
4) faire "Débogueur Windows local" pour créer le dossier "ide/ShoutEmUp/x64/Debug" (ignorer l'erreur, c'est normal)
5) Copier les fichiers "SDL2.dll" et SDL2_image.dll" de "SDL2/lib/x64" dans le dossier précédement créé
6) enjoy
